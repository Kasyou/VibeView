# ClaudeMD — Claude 开发经验知识库

## 这是什么

这个文件夹收集了用 Claude Code 做各种类型项目的实战经验。丢到新项目目录里，Claude 自动读取这些经验，知道怎么高效干活。

## 使用方式

1. 把这个文件夹复制到新项目目录
2. Claude 会自动读取 CLAUDE.md 和 knowledge/ 下的所有文件
3. 做项目时 Claude 会参考这些经验来工作

如果你有单片机/嵌入式相关的项目，另外还有一份 STM32 开发经验知识库可用。
